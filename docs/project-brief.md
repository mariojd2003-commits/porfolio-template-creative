## Project Concept

### What are you building?

Estoy desarrollando un portfolio web personal utilizando únicamente HTML, CSS y JavaScript nativo. El proyecto pone el foco en animaciones suaves, diseño responsive, modo claro/oscuro y una interfaz cuidada basada en variables y principios de design system

Está dirigido a desarrolladores, diseñadores y profesionales del sector digital que buscan una base sólida para crear su propio portfolio. Funciona como una plantilla reutilizable y adaptable, al mismo tiempo que sirve como carta de presentación para demostrar mis habilidades técnicas y criterio visual.

### Why does it matter?

Este proyecto me permite demostrar de forma práctica cómo estructuro, diseño y desarrollo una interfaz desde cero, cuidando tanto la experiencia de usuario como la consistencia visual. Además, sirve como base sólida sobre la que iterar y mejorar, integrando nuevas funcionalidades con la IA para aumentar mis conocimientos en desarrollo

## Design Direction

### Visual Style

El contenido visual se articula en torno a una estética oscura y tecnológica con temática morada, donde el elemento principal es un fondo interactivo generado mediante canvas, compuesto por una retícula de puntos que reaccionan al movimiento del cursor. Este sistema aporta dinamismo y profundidad, creando una sensación de entorno vivo y sensible a la interacción del usuario.

Sobre este fondo se superponen elementos tipográficos de gran peso visual, con tamaños jerárquicos claramente diferenciados que guían la atención. La composición mantiene un equilibrio entre minimalismo y expresividad, evitando la saturación visual y reforzando la legibilidad.

El uso de animaciones sutiles y transiciones suaves contribuye a una experiencia fluida, donde la interfaz no solo se observa, sino que se percibe como un sistema reactivo. En conjunto, el diseño busca transmitir una identidad contemporánea, centrada en la interacción, el control y la claridad visual.

## Reflection Questions

### What excites you most about this project?

Lo que más me motiva de este proyecto es el apartado visual y la capacidad de dar vida a la interfaz a través de animaciones e interacciones. Me interesa especialmente cómo el movimiento, los efectos y los pequeños detalles pueden transformar una web en una experiencia más atractiva y dinámica. Poder experimentar con este tipo de recursos me ha permitido construir algo que no se siente estático, sino más cercano a una experiencia interactiva.

### What challenges do you anticipate?

El mayor reto ha sido integrar las herramientas de IA con mi propio código y adaptar sus propuestas a mi sistema. En muchos casos, las soluciones generadas necesitaban ser reinterpretadas para encajar con mi estructura, especialmente en el uso de variables y estilos. Ajustar todo para que funcionara de forma coherente con los nuevos diseños ha sido un proceso exigente, pero clave para mantener consistencia en el proyecto.

### How does this project connect to your learning goals?

Este proyecto ha sido especialmente útil de cara a mi TFG, ya que comparte una lógica similar: construir un sistema de diseño basado en tokens a partir de un repositorio existente que no he desarrollado desde cero. Esto implica adaptarme a decisiones y estructuras previas, manteniendo coherencia con el sistema ya establecido. A lo largo del proceso, he podido comprender mejor cómo organizar variables, preservar la consistencia visual y evolucionar sistemas existentes para responder a nuevas necesidades, algo directamente aplicable a ese contexto.
